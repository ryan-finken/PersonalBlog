Below you'll find descriptions of a few projects I've created. Click the title to check out the source code on github!
Sports Scheduler
Built with Spring Boot, Thymeleaf, and Hibernate. An app to help recreational sports players, coaches, and leagues connect. Users can find, create, and register for events at their favorite venue. Learning about other players is as easy as viewing the user roster and clicking through to individual user profiles.
This blog!
Built with Spring Boot, Thymeleaf, and Hibernate. A simple blog site with time stamped posts persisted in a MySQL database.
